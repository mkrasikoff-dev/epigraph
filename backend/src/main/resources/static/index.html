О приложении

Как добавить цитаты из Яндекс Книг

Перейдите в раздел «Добавить» → «Импорт из JSON». Нужно создать файл со списком цитат в формате JSON. Подробная инструкция на экране импорта.

Хранение данных

Все цитаты хранятся в базе данных PostgreSQL и сохраняются между сессиями. Используйте экспорт для резервных копий.
